# Bubble Game

> A small browser-based bubble popping game built with HTML, CSS, and JavaScript.

## Demo
- Open `index.html` in your browser to play the game locally.

## How to play
- Click or tap the bubbles to pop them and score points.
- Try to pop as many bubbles as you can — difficulty may increase over time.

## Run locally
- Option 1: Open the file directly

  1. Double-click [index.html](index.html) or open it in your browser.

- Option 2: Serve with a simple HTTP server (recommended for some browsers)

  ```powershell
  python -m http.server 8000
  # then open http://localhost:8000 in your browser
  ```

## Project structure
- [index.html](index.html) — Main HTML file
- [style.css](style.css) — Styles for the game
- [script.js](script.js) — Game logic and interactions

## Customize
- Edit `script.js` to change spawn rate, bubble size, scoring, and behavior.
- Edit `style.css` to change visuals, colors, and layout.
